# Carpeta `assets/` (arrel del repo) — LEGACY

Aquesta carpeta conté una còpia antiga d’alguns CSS, JS i imatges que es feien servir
abans de migrar-ho tot a `docs/assets/`.

- La **carpeta d’assets oficial** del projecte és `docs/assets/`.
- Els nous materials (imatges, CSS, JS) **sempre** s’han d’afegir a `docs/assets/`.
- Els fitxers d’aquesta carpeta només es conserven per compatibilitat i com a arxiu històric.
  Es podran eliminar més endavant quan es revisin totes les referències.

Elements destacats:
- `assets/css/` i `assets/js/`: versions antigues dels fitxers que ara viuen a
  `docs/assets/css/` i `docs/assets/js/`.
- `assets/data/presentations.json`: catàleg global de presentacions **antic**.
  El catàleg oficial és `docs/presentacions/all.json`, generat per `scripts/build_catalog.py`.

> ℹ️ Si busques on afegir nous recursos compartits, vés a `docs/assets/`.
