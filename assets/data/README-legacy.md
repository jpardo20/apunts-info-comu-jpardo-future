# Catàleg de presentacions legacy

Aquest fitxer (`presentations.json`) és un antic catàleg global de presentacions
que es feia servir amb un script JavaScript (`assets/js/script.js`) en una versió
anterior del portal.

Actualment:

- El catàleg **oficial** de presentacions és `docs/presentacions/all.json`.
- `assets/js/script.js` i aquest fitxer es consideren **legacy** i no es fan servir
  en el flux actual de MkDocs.

Es manté per referència històrica fins que tot el contingut es mogui definitivament
a `docs/presentacions/all.json`.
